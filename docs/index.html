ARGENTOR

Secure multi-agent AI framework in Rust with WASM sandboxed plugins, MCP integration, and compliance modules.

Key Features

🔒

WASM Sandboxed Plugins

Skills run in WebAssembly (wasmtime + WASI) with capability-based permissions. No skill can escape its sandbox.

🤖

Multi-Agent Orchestration

Orchestrator-Workers pattern with specialized agents (Spec, Coder, Tester, Reviewer) following Anthropic's recommended design.

🔌

MCP Centralized Proxy

All tool calls routed through a central control plane with logging, metrics, rate limiting, and progressive tool disclosure.

🧠

Semantic Memory

Local embedding-based vector memory with hybrid BM25 + cosine similarity search. JSONL persistence.

⚖️

Compliance Built-in

GDPR, ISO 27001, ISO 42001, and DPGA modules. Audit logging, consent tracking, bias monitoring.

🛡️

Multi-Provider LLM

Claude, OpenAI, Gemini, OpenRouter, Groq, Ollama, and 7 more providers. Automatic failover across backends.

Quick Start

# Clone and build git clone https://github.com/fboiero/Argentor.git cd Argentor cargo build --workspace # Run the demo (no API keys needed) cargo run --example demo_pipeline # Start the gateway server cargo run --bin argentor -- serve # List skills cargo run --bin argentor -- skill list
